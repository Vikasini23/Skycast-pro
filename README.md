A high-performance, glassmorphism-style weather dashboard built with Vanilla JavaScript and WeatherAPI.

## Features
- **Auto-Location:** Uses Browser Geolocation API to fetch local weather instantly.
- **Real-time Data:** Provides temperature, wind speed, humidity, UV index, and AQI.
- **3-Day Forecast:** Dynamic rendering of upcoming weather conditions.
- **Smart UI:** Adaptive background colors for day/night cycles.
- **Persistent History:** Saves recent searches using `localStorage`.

## Tech Stack
- HTML5 / CSS3 (Flexbox & Grid)
- JavaScript (ES6+ / Async-Await)
- [WeatherAPI.com](https://www.weatherapi.com/)

## How to Run
1. Clone this repository.
2. Open `index.html` in any modern web browser.
3. Allow location access for the best experience.
